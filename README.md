# Event Management Web Application

This project is a web-based event management application developed as an academic project.  
The application focuses on managing event-related data and implementing basic web application concepts.

## Features
- Display event information
- Event data management
- Simple and user-friendly interface
- Web-based application structure

## Tech Stack
- HTML
- CSS
- JavaScript

## Purpose
This project was created for learning and academic purposes to practice web development fundamentals, including structuring web pages, styling, and basic interactivity.

## How to Run
1. Download or clone this repository
2. Open the project folder
3. Run the project using a local web server or open the main HTML file in a browser
